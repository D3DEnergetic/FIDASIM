FIDASIM

Note

This is the documentation for the release version of FIDASIM. Click here for the current development documentation.

FIDASIM is a fast Fortran 90 code that simulates neutral beam deposition and various fast-ion diagnostics in axisyimmetric fusion devices.

For more information, check out the user guide.
